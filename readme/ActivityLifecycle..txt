Activity LifeCycle

Activity 시작
1. onCreate : Activity 자동 생성
> 각종 이벤트 설정
2. onStart : Activity 시작
> 화면이 뜰때마다 해야되는 설정 (GPS나 유투브 프리미엄 같은 경우)
3. onResume : Activity가 사용자에게 보이기 시작할때
> 영상 처음눌렀을때 시작 등의 가벼운 데이터들을 임시로 저장하는 것의 작업들
4. onPause : Activity 일시정지(화면이 가려지거나 조금 투명하게 되었을 때)
> 전화오거나, 재난문자 등이 왔을 때
4-1. onPause -> onResume
> onPause에서 확인을 눌렀거나 할때(다시 화면이 전체 보일 때)
5. onStop : Activity가 정지 되었을 때 (화면이 완전이 
안보일때)
> 휴대폰 화면종료
> onStart와 반대로 앱 화면을 끄는 경우
5-1. onRestart(onStop -> onStart) : 화면이 다시 돌아왔을때
> 화면이 뒤로 갔다가 다시 진행하겠냐 등의 팝업 등의 경우
6. onDestroy : Activity(의 모든 데이터가) 없어짐 (뒤로가기 혹은 앱 데이터 삭제로 완전히 종료할때)
> 액티비티가 완전히 종료 되었을때
> 이제 사용하지 않을 파일혹은 앱 같은 것을 닫을 때 완전히 닫는 경우
00. 데이터가 부족하면 완전히 종료 된다.
01. 뒤로가기 했을 경우 onDestroy가 호출 될 가능성이 높음
02. 홈 키를 눌렀을 경우에는 onStop이 호출 될 수 있다.
03. 또한 동시에 호출 되는 경우가 있으니 이해하는 데에 주의바람!
