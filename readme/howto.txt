>> 카카오톡 더보기란 화면 구현 
먼저 1,2,3번 모두 반응형 앱요소(가로,세로에 따른 앱 화면)가 필요하기 때문에 콘스트레인트로 틀을 묶는다.

상단 : 콘스트더보기 텍스트 + 간단한 이미지 링크3개 ( 사용)
하단 : 광고는 이미지로 Data를 비워둔다.
리니어 레이아웃 (반복되는 것에서는 리니어 레이아웃이 사용하기 좋다)
중간에 노란색의 카카오페이 <지갑> 섹션은 콘스트레인트 레이아웃을 사용하여,
콘스트 레이아웃 속, 하단에는 리니어로 텍스트 3컨텐츠를 좌 앵커로 묶고, 우측 하단에는 우,좌,하단으로 콘스트 앵커를 묶어서 이미지+텍스트를 표현한다.

또한 중앙에 이모티콘 플러스 사용하기 등, 하얀 섹션은 중첩이 되는 섹션이니 Relative layout을 사용하여 중복으로 겹쳐 표현한다.(버튼 이벤트로 링크를 걸면 아주 좋을듯)

그리드 레이아웃으로 16개의  다양한 테이블을 표현한다.

실패 요인 : Bold 오류로 텍스트 굵기 조절 실패(해결)
