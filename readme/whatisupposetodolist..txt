# 2주차 과제 안내

💡 **액티비티 생명주기를 모두 "활용"하여 앱을 만들어 오세요.**

2주차 수업을 통해서 액티비티 생명주기란 무엇인지 학습하셨을 것 입니다.

여러분이 가장 먼저 선행을 하셔야 할 부분은 각 생명주기의 특징이 무엇인지, 각 생명주기의 활용을 어떻게 주로 하는지 또는 해야할지를 조사하시고 충분히 생각한 후에 여러분의 어플에 어떻게 녹여낼지를 기획해 주세요!

→ 개발일지를 통해 각 생명주기를 어떻게 활용할 것인지 기획한 내용과 그것을 진행한 과정, 구현하려던 기능에 대한 시행착오와 어떻게 해결하였는지를 적어주시면 좋을 것 같습니다!

**각각의 생명주기를 활용한 데는 이유가 전부 있어야 합니다. (중요 ⭐️⭐️⭐️)**

단순 OnCreate에서도 되고 onStart에서 해도 되는데 그냥 onCreate는 이미 썼으니 onStart에서 했어요 → 🙅‍♂️

다양한 플래그들 조사하면 과제에 도움 될 것 (ex. 아래와 같은 상황에는 어떤 일이 발생할까요?) 

핸드폰이 갑자기 꺼졌을때 전원버튼눌렀을때 전화가왔을 때 알람이 울렸을때 어떤 생명주기가 호출되는걸까?

> 개발자 문서의 내용을 참고해보며 생명주기에 대한 플래그와 그 내용에 대해 학습해보세요 🚴‍♀️🚴‍♀️
